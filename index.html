HTML and CSS basics summary

HTML summary

HTML (HyperText Markup Language) is used to define our contenct, structure and meaning.

You don't use it for styling purposes. Use CSS for that instead!

  • HTML uses "elements" to describe(annotate) content
  • HTML elements typically have an opening tag, content and a closing tag
  • You can also have empty(void) elements like images
  • You can also configure elements with attributes
  • There is a long list of attributes, but you will gain experience over time, no worries

Learn more about all the elements on the MDN HTML elements reference

CSS summary

CSS (Cascading Style Sheets) is used for styling your page content.

  • Styles are assigned with property-value pairs
  • You can assign styles via "style" attribute
  • To avoid duplication, you usualy use global "stles" - via the "style" element instead
  • Alternatively yo can work with external style file, which you "link" to
  • When working with CSS, things like "inheritance", "specificity" and "box model" should be understood

Learn more about the CSS properties and values on the MDN CSS reference
